📖 Dashboard Instructions

📌 KPI Definitions
Logged — Total cases registered in the selected Month & Year.
Closed — Cases logged AND closed within the same selected month.
Pending — Logged minus Closed for the selected month.
Old Closed — Cases closed this month but logged in a previous month.
Overall Pending — All unresolved cases cumulated up to the selected month.
🔽 How to Use the Filters
1
Month & Year — Select a month to view KPIs for that period. % badges compare to the previous month automatically.
2
Zone / Vendor / Store / Critical Product — All slicers work together and update every KPI and chart instantly.
3
Pending >30 Days chart and FM Closed >30 Days chart use frozen monthly snapshots — prior month values never change when new data is added.
